Notas importantes

  • Los datos quedan almacenados en la clave matidiaz_members dentro de chrome.storage.local.
  • No se envían emails ni datos a servidores externos desde esta pantalla.
  • Cada miembro obtiene una credencial automática con formato MATI-XXXXXX-AAAA; compartila de forma segura.
  • Si tu copia de md outbound se conecta a un servidor remoto, revisá los comentarios en el código para adaptar cualquier sincronización necesaria.
